Mosaicos con GDAL
¶

GDAL es una biblioteca traductora para formatos de datos geoespaciales, que adems proporciona un conjunto de herramientas para el tratamiento de estos datos. Se publica bajo una licencia de cdigo abierto de estilo MIT por parte de la Fundacin Geoespacial de Cdigo Abierto (OSGEO). Existen varias motivos por las cuales GDAL es una de las mejores opciones para generar mosaicos:

  • Soporte de formatos: Es capaz de leer y escribir una amplia gama de formatos de datos geoespaciales, como GeoTIFF, JPEG, PNG, MrSID, ECW, entre otros. Esto significa que puedes utilizar imgenes en diferentes formatos y combinarlas en un solo mosaico sin tener que convertirlas previamente a un formato especfico.

  • Eficiencia y rendimiento: Est optimizado para el procesamiento eficiente de datos geoespaciales y mosaicos de imgenes. Utiliza tcnicas avanzadas de acceso y manipulacin de datos, lo que permite manejar grandes conjuntos de imgenes de manera eficiente y rpida.

  • Compatibilidad con diferentes plataformas: Es compatible con varias plataformas, incluyendo Windows, macOS y Linux. Esto brinda la flexibilidad de trabajar en diferentes entornos de sistema operativo y utilizar GDAL en diversas configuraciones y herramientas.

  • Comunidad y documentacin: Cuenta con una amplia comunidad de usuarios y desarrolladores que brindan soporte y comparten conocimientos. Adems, hay una documentacin detallada disponible, lo que facilita el aprendizaje y la resolucin de problemas.

4. Creación de mosaico con gdalbuildvrt¶

gdalbuildvrt es otra potente herramienta proporcionada por GDAL. Este comando se utiliza para construir un archivo Raster Virtual (VRT) que es un archivo XML que define una coleccin virtual de conjuntos de datos raster.

Esta herramienta construye un VRT a partir de una lista de conjuntos de datos de entrada. La lista de conjuntos de datos se puede especificar al final de la lnea de comando, o colocarse en un archivo de texto para listas muy largas.

El archivo VRT acta como un mosaico virtual que proporciona acceso y vista previa a mltiples conjuntos de datos raster sin tener que fusionarlos fsicamente en un solo archivo. Esto permite trabajar con grandes conjuntos de datos raster de manera eficiente y sin necesidad de copiar o alterar los archivos originales.

Para este ejercicio utilizaremos un archivo de texto con el nombre de los archivos de entrada que formaran el mosaico. Vamos a construir este archivo utilizando el comando dir con un patron que liste solo los archivos .tif.
